# Frontend

Сделана реализация двух страниц по роутам:

- ```/``` - главная страница со списком игр
- ```/game/:id``` - страница игры с подробным описанием

## Технологии

- typescript v5
- react v18.2
- redux toolkit
- react-router-dom v6
- axios
- sass
- eslint

## Инструкция по запуску

Установите зависимости

```
npm install
```

Запустите проект в dev-режиме

```
npm run start
```
